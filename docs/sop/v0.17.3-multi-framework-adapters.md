# SOP: v0.17.3 — Multi-Framework Adapters

> Purpose: add framework-specific SSR adapters for vanilla Web Components
> and React-to-Web-Component bridging, expanding LessJS beyond Lit.
>
> Release type: **minor (new packages, no breaking change)**.

## Goal

1. **adapter-vanilla**: SSR adapter for plain/vanilla Web Components whose
   `render()` returns a string directly (no template framework). Provides
   DSD hydration base class and auto-registration.
2. **adapter-react**: SSR adapter for React components wrapped as Web
   Components. Handles `React.createElement` → string conversion for DSD
   rendering without a DOM.

### Implementation Details

#### Scope 1: adapter-vanilla

Plain Web Components that `render(): string` already work with core's
`renderDSD()` — the adapter adds:

- `installVanillaAdapter()`: registers `'vanilla'` adapter via
  `registerAdapter()`. Detects string render output, provides
  `extractStyles()` for components with static `styles` property.
- `DsdVanillaElement`: base class like `DsdLitElement` but for vanilla
  components (extends `HTMLElement`). Provides DSD detection + hydration
  event binding + cleanup.
- Zero framework deps — pure DOM API.

Package structure:

```
packages/adapter-vanilla/
  deno.json
  src/
    index.ts          — public exports
    ssr.ts            — installVanillaAdapter + extractVanillaStyles
    dsd-hydration.ts  — DsdVanillaElement + WithDsdHydration mixin
  __tests__/
    ssr.test.ts
    dsd-hydration.test.ts
  README.md
  LICENSE
```

#### Scope 2: adapter-react

Renders React components to HTML strings for DSD during SSR:

- `installReactAdapter()`: registers `'react'` adapter. Uses
  `ReactDOMServer.renderToStaticMarkup()` for string conversion.
- `DsdReactElement`: helper that wraps a React component as a Web
  Component. On SSR, the adapter renders the React tree to string.
  On client, `ReactDOM.createRoot()` mounts into shadow DOM.
- Depends on `react` and `react-dom` as peer deps (npm:).

Package structure:

```
packages/adapter-react/
  deno.json
  src/
    index.ts          — public exports
    ssr.ts            — installReactAdapter + React SSR rendering
    dsd-hydration.ts  — DsdReactElement wrapper
  __tests__/
    ssr.test.ts
    dsd-hydration.test.ts
  README.md
  LICENSE
```
